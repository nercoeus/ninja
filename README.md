# 不逝之影（ninja）

### 在巨人实习期间参加巨人校招游戏 demo 大赛的作品（由对不队合力制作 5 人）

为期 1 个月，后端使用 zero 库，实现简单的数据传输。前端使用 unity-3d 来进行游戏的实现，最后获得第一名的游戏作品。
